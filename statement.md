# 📑 PROJECT STATEMENT
# 🎯 <u>PROBLEM STATEMENT</u>
Many students struggle to track if their study hours and attendance are enough to pass. This project uses AI (Decision Trees) to predict academic success and provide an early-warning system for at-risk students.

# 🔭 <u>PROJECT SCOPE</u>
Prediction: Classifying students as Pass or At Risk.

Analysis: Checking impact of attendance on final results.

Advice: Providing automated study recommendations.

Validation: Ensuring inputs are within logical limits.

# 👥 <u>TARGET USERS</u>
🎓 Students: For self-assessment and habit tracking.

👨‍🏫 Mentors: To identify and help struggling students.

🏫 Institutions: To monitor and improve pass percentages.

# 🚀 <u>HIGH-LEVEL FEATURES</u>
🤖 ML Engine: Built using Scikit-learn for accurate predictions.

🏗️ Modular Code: Separate files for Preprocessing, Prediction, and UI.

🛡️ Error Handling: Prevents crashes from invalid data inputs.

📊 Live Feedback: Instant terminal-based status reports.
